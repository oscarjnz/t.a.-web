Generación de Código

Código Intermedio

El código intermedio es una representación de bajo nivel del código fuente, independiente de la máquina objetivo. Los formatos incluyen Código de Tres Direcciones (ej., t1 = a + b), Bytecode, SSA y AST Anotado. Ventajas: portabilidad, optimización más sencilla, independencia de la arquitectura.

Ejemplo: Código fuente a = b + c * d a Código de Tres Direcciones: t1 = c * d; t2 = b + t1; a = t2

Código Final

El código final es la salida ejecutable, como código máquina o ensamblador. El proceso implica selección de instrucciones, asignación de registros y planificación.

Ejemplo: Código de Tres Direcciones t1 = a + b a Ensamblador: ADD R1, a, b; MOV t1, R1
